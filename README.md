## 🛠 Collaboration Rules (Read before pushing!)

### 1. Naming Convention
*   Use **lowercase** for all filenames (e.g., `services.html`, NOT `Services.html`).
*   Save all images in the `/images` folder.

### 2. Git Workflow
*   **Do not** push directly to the `main` branch.
*   Create a new branch for your work: `git checkout -b feature-yourname`
*   Once finished, push your branch and open a **Pull Request (PR)** for the team to review.

### 3. HTML Standards
*   Use **Semantic Tags**: `<header>`, `<nav>`, `<main>`, `<section>`, and `<footer>`.
*   Include a `<title>` and `<meta name="description">` on your page for **SEO**.
*   Ensure your page has a link back to `index.html` in the navigation.

---

## 🚀 How to Get Started
1. Clone the repo: 
   `git clone [URL-TO-THIS-REPO]`
2. Create your page using the filename assigned above.
3. Link your page to the navigation menu found in `index.html`.
